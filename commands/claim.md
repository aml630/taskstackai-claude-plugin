---
description: Claim an issue to work on
argument-hint: <id>
---

Claim an issue for work. This assigns the issue to you and sets its status to in_progress.

If no issue ID is provided, ask for one.

Run:
```
${CLAUDE_PLUGIN_DIR}/bin/taskstackai claim <id>
```

Confirm the claim after the command completes. Suggest the user can now start working and use `/close` when done.
